# Chez Liliane (Lilian's Bakery)

A modern, responsive portfolio website for Lilian's Bakery, showcasing baking creations with a premium visual design.

## Project Overview

This is a **Next.js 16** project built with **TypeScript** and **Tailwind CSS**. It serves as a personal brand website featuring:

- **Portfolio Showcase**: High-quality gallery of baking works (cakes, breads, pizzas).
- **Dark Mode Support**: Automatically adapts to system preferences with optimized contrast.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **Icons**: React Icons
- **Deployment**: Vercel

## Getting Started

1. **Install dependencies**:
   ```bash
   pnpm install
   ```

2. **Run development server**:
   ```bash
   pnpm dev
   ```

3. **Open in browser**:
   Navigate to [http://localhost:3000](http://localhost:3000).

## Deployment

This project is deployed on **Vercel**.

To deploy manually:
```bash
git push
```
(Vercel will automatically trigger a new build on push to `main`)
